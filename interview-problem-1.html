多个有序数组求最小范围

有3个有序的数组，请找到一个最小的数字范围。使得这k个有序数组中，每个数组都至少有一个数字在该范围中。

例如：

[4, 10, 15, 24, 26]

[0, 9, 12, 20]

[5, 18, 22, 30]

所得最小范围为[20,24]，其中，20在2中，22在3中，24在1中。

/**
 * Created by zero on 2015/11/14.
 * SimpleCodeDemo
 *
 * 问题：
 * k个有序数组求最小范围,每个数组都至少有一个数字在该范围中.
 *
 * 分析：
 * 最终得到的最小的范围，至少包含三个元素，并且在所有数组整体的排序中，是相邻的。
 * 假设最小范围是[a, b, c]，a < b < c，并且a, b, c来自不同的数组。
 * 还有一种情况[a, b, d, c]，中间有一个d，a < d < c，此时d只能是和b一个数组。
 *  因为：1.若d和a一个数组，那么区间即为[b, d, c]
 *       2.若d和c一个数组，那么区间即为[a, b, d]
 *       3.d和b一个数组，区间范围大小不变，考虑b或d都没有影响
 *
 * 因此，只需要考虑在最终的排序中，考虑邻近的、并且来自不同有序数组的元素，作为备选的范围。
 * 利用归并排序的思想，用三个指针指向三个数组的第一个元素。
 * 确保每次都是3个来自不同的数组的元素进行比较，得到最大值、最小值。
 * 记录最小区间，让元素值最小的指针后移一位，循环。
 * 直到有一个数组遍历结束，记录得到的区间即为所求。
 *
 */
public class MinRangeOfList {

    /**
     * 求3个数组的最小范围
     *
     * @param a, b, c 给定的三个有序数组
     * @return 长度为2的int数组，表示左右区间
     */
    public static int[] mindiff(int a[], int b[], int c[]) {
        if (a == null || b == null | c == null) return null;
        if (a.length <= 0 || b.length <= 0 || c.length <= 0) return null;

        int i = 0, j = 0, k = 0;//分别指向三个数组当前元素
        int sizeA = a.length, sizeB = b.length, sizeC = c.length;

        int max, min;
        //初始化区间为所有元素的最小值到所有元素的最大值
        int[] diff = {getMin(a[0], b[0], c[0]), getMax(a[sizeA - 1], b[sizeB - 1], c[sizeC - 1])};

        while (i < sizeA && j < sizeB && k < sizeC) {
            System.out.println(String.format("a:%d\tb:%d\tc:%d", a[i], b[j], c[k]));

            //三个来自不同数组的元素进行比较，得到区间范围
            max = getMax(a[i], b[j], c[k]);
            min = getMin(a[i], b[j], c[k]);

            //最小元素的指针后移
            if (min == a[i]) i++;
            else if (min == b[j]) j++;
            else k++;
            System.out.println(String.format("max:%d\tmin:%d", max, min));

            //记录最小区间
            if (max - min < diff[1] - diff[0]) {
                diff[1] = max;
                diff[0] = min;
            }
        }

        System.out.println(String.format("=> [%d, %d]", diff[0], diff[1]));
        return diff;
    }

    private static int getMax(int a, int b, int c) {
        return Math.max(Math.max(a, b), c);
    }

    private static int getMin(int a, int b, int c) {
        return Math.min(Math.min(a, b), c);
    }

    public static void main(String[] args) {
        int[] a = {4, 10, 15, 24, 26};
        int[] b = {0, 9, 12, 20};
        int[] c = {5, 18, 22, 30};

        int[] diff = mindiff(a, b, c);
    }
}

找出数组中是否有重复的数

int数组a[N]，数组中的元素都在[0,N)的范围内，写出一个算法，判断如果有重复元素，任意返回一个即可，如果没有重复元素，返回-1. 要求：空间复杂度O(1), 时间复杂度O(N).

/**
 * Created by zero on 2015/11/23.
 * SimpleCodeDemo
 */
public class CheckDuplicate {

    /**
     * 最简单粗暴的方案，双重循环寻找重复元素
     * 
     * 空间复杂度O(1), 时间复杂度O(N^2)
     */
    public static int check1(int[] a, int N) {
        for (int i = 0; i < N; i++) {
            for (int j = i + 1; j < N; j++) {
                if (a[i] == a[j]) return a[i];
            }
        }
        return -1;
    }

    /**
     * 标志数组法
     *
     * 申请长度为N的辅助布尔型数组，作为标志位
     * 将数组中元素值作为索引，访问布尔数组并修改为true
     * 访问布尔数组遇到true时，即找到重复元素
     *
     * 空间复杂度O(N), 时间复杂度O(N)
     */
    public static int check2(int[] a, int N) {
        boolean[] aux = new boolean[N];
        //初始化全部置为false
        for (int i = 0; i < N; i++)
            aux[i] = false;

        //将当前元素值所指向的布尔值设置为true，再次遇到true，即重复
        for (int i = 0; i < N; i++) {
            if (aux[a[i]]) return a[i];
            aux[a[i]] = true;
        }

        return -1;
    }

    /**
     * 固定偏移标志法
     * 
     * a[N]，里面是0至N-1。
     * 原数组a[i]最大是N-1，若a[i]=K出现后，将a[K]减一次N，必然会小于0。
     * 当某处a[i]=K再次成立时，查看a[K]即可知道K已经出现过。
     * 
     * Java中int范围：-2147483648 到2147483648，a[i] - N 并不会产生溢出问题。
     * 
     * 空间复杂度O(1), 时间复杂度O(N)
     */
    public static int check(int[] a, int N) {
        int K;
        for (int i = 0; i < N; i++) {
            //拿到a[i]位置的实际值
            if (a[i] < 0) K = a[i] + N;
            else K = a[i];

            if (a[K] >= 0) a[K] -= N;//a[K]不小于0，即a[i]未出现过，给a[K]做上标记
            else return K;//a[K]小于0，即a[i]出现过，返回
        }
        return -1;
    }

    public static void main(String[] args) {
        final int N = 5;
        int[] a = {2, 1, 3, 1, 4};
        System.out.println(check(a, N));
    }
}

广度遍历非完全二叉树

给定一个二叉树，广度遍历，按层打印出每个节点的值。

/**
 * Created by zero on 2015/11/13.
 * SimpleCodeDemo
 *
 * 二叉树
 *
 * 广度优先遍历 -> 队列
 * 深度优先遍历 -> 栈
 */
public class BinaryTree {

    private class Node {
        int val;
        Node left;
        Node right;

        public Node(int val) {
            this.val = val;
        }
    }

    private Node root;//根节点

    public BinaryTree(int[] arr) {
        root = createTreeByArray(arr, 1);
    }

    /**
     * 递归创建二叉树
     *
     * @param arr 二叉树的数组表示法
     */
    private Node createTreeByArray(int[] arr, int index) {
        if (arr != null && arr.length > 0 && index < arr.length) {
            int value = arr[index];
            if (value != 0) {
                Node node = new Node(value);
                arr[index] = 0;
                node.left = createTreeByArray(arr, index * 2);
                node.right = createTreeByArray(arr, index * 2 + 1);
                return node;
            }
        }
        return null;
    }

    /**
     * 广度优先遍历
     * 采用队列存储节点
     */
    public void levelOrderTraversal() {
        if (root == null) {
            System.out.println("Tree empty!");
            return;
        }

        Queue<Node> queue = new ArrayDeque<>();//队列存储节点
        queue.add(root);

        while (!queue.isEmpty()) {
            Node node = queue.remove();
            System.out.print(node.val + " ");

            if (node.left != null) queue.add(node.left);//下一层节点入队
            if (node.right != null) queue.add(node.right);
        }

        System.out.println("\n");
    }

    /**
     * 深度有限遍历,相当于前序遍历
     * 采用栈存储节点
     */
    public void depthOrderTraversal() {
        if (root == null) {
            System.out.println("Tree empty!");
            return;
        }

        Stack<Node> stack = new Stack<>();//栈存储节点
        stack.push(root);

        while (!stack.empty()) {
            Node node = stack.pop();
            System.out.print(node.val + " ");

            if (node.right != null) stack.push(node.right);
            if (node.left != null) stack.push(node.left);
        }

        System.out.println();
    }

    /**
     *          9
     *        /  \
     *       3    2
     *     /  \    \
     *    5    7   12
     *    \   / \   \
     *     9 5   6   8
     * {0,9,3,2,5,7,0,12,0,9,5,6,0,0,0,8}
     *
     * 二叉树数组表示法，首位为0，从1开始，null节点记为0
     *
     */
    public static void main(String[] args) {
        BinaryTree tree = new BinaryTree(new int[]{0,9,3,2,5,7,0,12,0,9,5,6,0,0,0,8});

        tree.levelOrderTraversal();//结果：9 3 2 5 7 12 9 5 6 8

        tree.depthOrderTraversal();//结果：9 3 5 9 7 5 6 2 12 8 
    }
}

毒药和小白鼠的问题

有10瓶一模一样的瓶子，其中9瓶普通的水，1瓶是毒药，任何喝下毒药的生物都会在24小时之后死亡。 现在，你只有4只小白鼠和24小时的时间，如何检验出哪个瓶子里有毒药？

利用二进制的思想：

0000 = 0

0001 = 1

0010 = 2

0011 = 3

0100 = 4

0101 = 5

0110 = 6

0111 = 7

1000 = 8

1001 = 9


一个小白鼠代表一位二进制，0-9 号表示10个瓶子。

将1,3,5,7,9号瓶子的水混起来给1号老鼠喝

将2,3,6,7号瓶子的水混起来给2号老鼠喝

将4,5,6,7号瓶子的水混起来给3号老鼠喝

将8,9号瓶子的水混起来给4号老鼠喝

哪个老鼠死了，相应的二进制位标为1.

如1号老鼠死了，2号老鼠没死，3号老鼠死了，4号老鼠没死，那么结果就是0101=5，就是5号瓶的水是毒药。
